Q1) Given a matrix [[a, b], [c, d]], recall that one can calculate its eigenvalues by solving the characteristic polynomial λ^2 - (a+d)+(ad−bc)=0. In this quiz, you will practice calculating and solving the characteristic polynomial to find the eigenvalues of simple matrices.
For the matrix A = [ [1, 0], [0, 2]], what is the characteristic polynomial, and the solutions to the characteristic polynomial?
A1) λ^2 −3λ+2=0
​	λ1 = 1, λ2 = 2

Q2) Recall that for a matrix A, the eigenvectors of the matrix are vectors for which applying the matrix transformation is the same as scaling by some constant. For A = [[1, 0], [0, 2]] as immediately above, select all eigenvectors of this matrix.
A2) [0, 2]
	[1, 0]
	[0, 3]
